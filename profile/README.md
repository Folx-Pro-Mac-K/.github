## 💡 Overview

<a href="#folxPro">Folx Pro</a> is a comprehensive download manager and torrent client for Mac — replacing the browser's basic download handling with an organized, accelerated download system that manages both HTTP/HTTPS downloads and BitTorrent with a unified interface and intelligent file organization.

The <a href="#folxPro">split-thread download acceleration</a> divides each file download into multiple simultaneous segments — downloading different parts of the file in parallel from the server and assembling them into the complete file on completion. This multi-thread approach saturates available bandwidth more completely than a single-connection download, delivering faster completion times particularly for large file downloads where connection efficiency matters most. The <a href="#folxPro">built-in torrent client</a> manages BitTorrent downloads alongside regular HTTP downloads in the same Folx interface — adding torrent files or magnet links, configuring upload and download speed limits per torrent, and managing the seeding behavior after download completion. No separate torrent application is needed alongside Folx for complete download coverage.

The <a href="#folxPro">tag-based organization system</a> applies user-defined tags to downloads — categorizing files by type, project, source, or any meaningful label for locating downloaded files later without navigating the file system. Downloads receive tags automatically based on configurable rules or manually during download management, and the Folx library shows all downloads filtered by any combination of tags. The <a href="#folxPro">download scheduling</a> queues downloads for specific times — scheduling large downloads for overnight or off-peak hours when bandwidth is less contested, setting start times for downloads that should begin at a specific moment, and managing the download queue to avoid simultaneous large downloads that would compete for bandwidth.</p>

<a href="#folxPro">Speed throttling controls</a> set upload and download speed limits — preventing Folx downloads from consuming all available bandwidth and leaving network capacity for other applications and users, particularly important for torrent seeding that can otherwise saturate upload bandwidth indefinitely. <a href="#folxPro">Proxy and VPN integration</a> routes downloads through configured proxy servers and VPN connections — applying network privacy settings to download traffic for users who require network privacy for their downloading activity.

<a href="#folxPro">Browser integration extensions</a> intercept browser download links and route them through Folx rather than the browser's native downloader — capturing downloads from any web source automatically without manually copying and pasting URLs into Folx. <a href="#folxPro">Music tagging and iTunes integration</a> identifies and tags music file downloads automatically — applying artist, album, track, and genre metadata to downloaded audio files and adding them to the music library without manual tagging.</p>
